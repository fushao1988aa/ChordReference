# ChordReference – 和弦调性速查器

**ChordReference** 是一款专为音乐制作人、作曲家和音乐学习者设计的智能和声助手插件（VST3，后续计划支持 AU）。它深度集成在宿主软件中，实时识别输入的 MIDI 和弦，自动分析调性，动态显示主(T)、下属(S)、属(D)功能的和弦卡片，并提供智能和弦进行生成、常用和弦进行预设浏览器、负和声映射、多主题皮肤等独特功能。

无论是快速试写和弦进行、学习调性和声，还是在创作中获取灵感，ChordReference 都能成为您音乐工作流中的得力工具。

---

## 核心功能

### 实时 MIDI 和弦识别与调性分析

- 支持从 MIDI 键盘或 DAW 钢琴卷帘输入和弦（3～7 个音符）
- 基于预构建的和弦数据库（含转位、引申音、增/减/挂留等质量）实时匹配
- 智能防抖算法避免识别抖动，优先匹配当前调性
- 识别结果即时反映在 T/S/D 卡片上，并高亮迷你键盘对应音

### T/S/D 功能和弦卡片

- 三张卡片分别显示当前调性的**主和弦(T)**、**下属和弦(S)**、**属和弦(D)**
- 支持 **3/7/9/11/13 扩展和弦**一键切换
- 支持**和弦质量切换**（大三、小三、属七、减三、增三、挂二、挂四、小大七、半减七）
- 支持**转位切换**（原位/一转/二转等）
- 引申音子菜单（属七和弦可选 b9, #9, #11, b13 及其组合）
- **拖拽生成 MIDI 文件**：将当前和弦拖入 DAW 轨道，自动生成一小节 MIDI 片段
- **内置试听**：点击卡片空白区域试听和弦（内置钢琴采样或外部 MIDI 输出）
- **情绪色彩显示**：根据调性主音与功能符号动态显示情绪描述和背景氛围色

### 五度圈与智能生成

- 内外双圈五度圈：外圈大调、内圈小调，点击直接切换调性
- 中心骰子：左键随机生成一组 T/S/D 和弦进行，右键弹出风格菜单（通用/古典、流行/摇滚、爵士/离调、古风/中国风）
- 生成的进行自动应用到 T/S/D 卡片，支持多利亚、弗里吉亚等 7 种调式

### 和弦进行预设浏览器

- **超过 700+ 个出厂预设**（华语流行、日系动漫、古风、爵士、布鲁斯、拉丁等）
- **分类筛选**：按调式/地域/情感/复杂度/曲式功能/中国风特别标签分类
- **标签过滤**：多标签组合过滤，实时显示匹配数量
- **搜索**：按预设名称、标签、描述搜索
- **收藏与常用**：标记常用预设，快速访问
- **用户自定义预设**：支持添加、编辑、删除、导入/导出（JSON 或 ZIP 包）
- **封面图片**：每个预设可自定义封面（Base64 存储，最大 180px）
- **拖拽生成 MIDI**：将整个进行拖入 DAW 轨道，生成完整的 MIDI 文件
- **试听播放**：选中预设后自动循环播放进行
- **用户文件夹**：可创建自定义文件夹，将用户预设分类存储
- **和弦进行分解**：以评分机制建立的非常庞大的和弦进行分解依赖库，多达300余分解范式
- **旋律生成**：目前正在开发中......

### 负和声模式

- 一键开启负和声映射，T/S/D 卡片自动翻转显示负和声版本的和弦
- 试听和 MIDI 拖拽均使用负和声版本

### 多主题皮肤与界面缩放

- 四种精美主题：现代紫、未来蓝、象牙米、尊黑金
- 所有 UI 元素自动适配皮肤
- 窗口自由缩放，内容使用 FlexBox 弹性布局自适应

### 中英文双语界面

- 完整支持中文/英文一键切换

### 注册与试用管理

- 30 天全功能试用期

## 技术亮点

- **高性能实时识别**：基于预计算和弦数据库，匹配速度快，CPU 占用极低
- **现代 C++ 与 JUCE 框架**：代码结构清晰，易于维护和扩展
- **跨平台支持**：Windows / macOS（AU 待定），VST3 格式
- **完全的 MIDI 拖拽支持**：生成的 MIDI 文件可直接拖入任何 DAW
- **强大和弦进行分解支持**：生成的 和弦进行分解MIDI 文件可直接拖入任何 DAW

## 构建与依赖

### 要求

- C++17 编译器（Visual Studio 2022 / Xcode / GCC）
- [VST3 SDK](https://steinbergmedia.github.io/vst3_dev_portal/)（需自行下载，放置于 `./VST_SDK/` 目录）
- [JUCE](https://juce.com/) 8.x 或更高版本（使用 Projucer 管理项目）

### 构建步骤

1. 克隆本仓库
2. 下载 VST3 SDK 并设置项目中的 SDK 路径
3. 打开 `ChordReference.jucer`（或对应的 IDE 解决方案文件）
4. 使用 Projucer 导出工程并编译
5. 生成的 VST3 插件位于 `Builds/` 目录下

> 注意：由于许可证限制，本仓库不包含 VST3 SDK 和 JUCE 二进制文件，请自行下载并配置。
